（一）学业篇
  刚刚从高中毕业的你曾梦想仗剑走天涯，看一看世界的繁华。没想到上了大学发现还是有一堆卷王，早早的知道各种消息，将你甩在身后。
  （能不能别卷了啊喂）
  成绩在大学有着另一种表述方式，叫绩点。所谓绩点，就是衡量一门课学的好不好的指标。满绩表示你在这门课拿到了所有分数。在hdu，拿到95分即可获得满绩，也就是5.0，由于四舍五入的存在，所以理论上一门课的总评达到94.5就可以取得满绩。每少考一分，绩点就会下降0.1，相应的94就是4.9绩点，93就是4.8绩点，以此类推，60分是1.5绩点。如果总评四舍五入没有到60分就会挂科。挂科的同学会在新学期初的时候进行补考，补考超过60分就不用重新修读这门课了，但是还是会有补考记录。（注意补考是没有平时分的，也就是说补考需要卷面60分，还是比较困难的）如果补考没到60分就要重修了，需要重新选课重新交学费。
  那么有人就要问了，平时分是个啥玩意？诚然，刚进大学的i人一听这个名词天都塌了。但是这就是事实，只能受着了。一门课的总评由平时分和期末考试分组成，有些科目是平时分期末分三七开（总评=平时分×0.3+期末分×0.7），有些是四六开，有的没有期末考的课程可能就把大作业成绩当成期末考的成绩。一些理学院开的课程，比如说高等数学、大学物理，还会有期中考试，那总评就会由三部分组成。（for example，总评=平时分×0.25+期中考成绩×0.15+期末考成绩×0.6）
  如何获得高的平时分呢？这个因老师而异。有的会要求发言，有的会要求做课前汇报（也有的老师把这个叫pre），有的会看实践作业，也有的会看课堂测成绩。这部分内容很难通过一段话写出来，所以大家选课之前最好先调研一下老师的标准。
  这里还需要补充一个很重要的概念叫学分。学分是用来衡量一门课重要程度很好的标准。比如说高等数学A1是5学分，但是形势与政策就只有0.25学分。我们最后的平均学分绩点是每门课的学分*每门课的绩点/所有课的学分加起来。所以学分高的课程对最终绩点的影响就会特别大，也会对重修需要交的学分影响特别大，重修一学分需要补交120元。Therefore，好好学吧。
  上面那段说到的平均学分绩点也叫裸绩点，还有一个概念叫GPA。打比赛可以快速获得GPA，比如说拿到一个A类甲等国家一等奖就可以在裸绩点的基础上+1.5，国二是+1.0，国三/省一是+0.7，省二是+0.5，省三是+0.3.除此之外还有奖金可以拿，美美得吃了。A类乙等也可以加GPA，但是没有奖金；B类甲等GPA会降级（国家级按照省级），并且也没有奖金。打比赛还可以获得创新学分，每个同学需要2分的创新学分才能毕业，所以大家尽量早点拿到，不要影响毕业了。竞赛的相关内容会在（二）里详细展开。
  那么这个GPA有啥用嘞？当然是评奖学金。GPA会和德智体美劳等元素加起来组成综测，综测从夯到拉排名就是奖学金的顺序。综测文件因学院而异。
  每个学期都会评优秀学生奖学金，大致可以分成一等奖学金、二等奖学金、三等奖学金、单项奖学金。体测不过和挂科（补考过也没用）的同学都评不了。
  一等奖学金是1500元，占专业人数的5%，要求平均学分绩点排名列专业前10%，且学分绩点大于等于4
  二等奖学金是750元，占专业人数10%，要求平均学分绩点排名列专业的25%，且学分绩点大于等于3.5
  三等奖学金是400元，占专业人数15%，要求平均学分绩点排名列专业的45%，且学分绩点大于等于2.5
  单项奖学金包括学习进步奖、学生工作奖、文艺活动奖、道德风尚奖、社会实践奖、创新创业奖、体育活动奖等。
  然后每年都会评选省政府奖学金（6000元）。一般来说拿两次一等奖学金是很稳的，拿一次一等、一次二等也有一定机会。
  每年也会评选国家奖学金（10000元）。这个就比较困难了，一个学院也没有几个名额。如果你读大学的时候发现自己好像还不错，很有天赋，就可以抓好绩点的同时多搞一搞科研竞赛、社会实践。如果能拿到国家奖学金的话，保研考研申请学校、就业的时候都会有质的飞跃。
  除此之外还有国家励志奖学金，这个主要面向家庭情况没有那么好的同学。只要你愿意努力，国家就会帮助你！
  如果你的绩点还不错，就可以进行转专业。（只能转一次，所以想好再转！）这个每年政策都不太一样，在2025级应该是大一下的时候可以进行第一次跨学院的转专业。文科专业转理工科专业时，如果高考没有选择物理，就需要参加物理考试。
  保研名额大概会在大三下的时候确认。这个老学长也不知道要怎么战胜，这里就不献上焚决了。但是需要注意的是，保研只是给你这个名额，具体去哪个学校得你自己去联系好导师（也叫套磁）、自己去参加夏令营、自己去预推免。这个如果学长能上岸的话后续会更新的。直博的意思是直接攻读博士，好处是只需要读五年就可以拿到博士学院，坏处了如果延毕了或者不给你毕业的话那就只有本科学历了。
