（零）zmy的话
  刚进电专的你面对陌生的环境、一大堆生僻的名词，非常之绝望！为了让大家快速摆脱SHEEP阶段，几个饱经风霜的老学长献上了自己的焚决。本焚决只是一份说明书，不是一个引路牌。每个人都有自己的路，走好自己的路才是最重要的。

（一）学业篇
  刚刚从高中毕业的你曾梦想仗剑走天涯，看一看世界的繁华。没想到上了大学发现还是有一堆卷王，早早的知道各种消息，将你甩在身后。
  （能不能别卷了啊喂）
  成绩在大学有着另一种表述方式，叫绩点。所谓绩点，就是衡量一门课学的好不好的指标。满绩表示你在这门课拿到了所有分数。在hdu，拿到95分即可获得满绩，也就是5.0，由于四舍五入的存在，所以理论上一门课的总评达到94.5就可以取得满绩。每少考一分，绩点就会下降0.1，相应的94就是4.9绩点，93就是4.8绩点，以此类推，60分是1.5绩点。如果总评四舍五入没有到60分就会挂科。挂科的同学会在新学期初的时候进行补考，补考超过60分就不用重新修读这门课了，但是还是会有补考记录。（注意补考是没有平时分的，也就是说补考需要卷面60分，还是比较困难的）如果补考没到60分就要重修了，需要重新选课重新交学费。
  那么有人就要问了，平时分是个啥玩意？诚然，刚进大学的i人一听这个名词天都塌了。但是这就是事实，只能受着了。一门课的总评由平时分和期末考试分组成，有些科目是平时分期末分三七开（总评=平时分×0.3+期末分×0.7），有些是四六开，有的没有期末考的课程可能就把大作业成绩当成期末考的成绩。一些理学院开的课程，比如说高等数学、大学物理，还会有期中考试，那总评就会由三部分组成。（for example，总评=平时分×0.25+期中考成绩×0.15+期末考成绩×0.6）
  如何获得高的平时分呢？这个因老师而异。有的会要求发言，有的会要求做课前汇报（也有的老师把这个叫pre），有的会看实践作业，也有的会看课堂测成绩。这部分内容很难通过一段话写出来，所以大家选课之前最好先调研一下老师的标准。
  这里还需要补充一个很重要的概念叫学分。学分是用来衡量一门课重要程度很好的标准。比如说高等数学A1是5学分，但是形势与政策就只有0.25学分。我们最后的平均学分绩点是每门课的学分*每门课的绩点/所有课的学分加起来。所以学分高的课程对最终绩点的影响就会特别大，也会对重修需要交的学分影响特别大，重修一学分需要补交120元。Therefore，好好学吧。
  上面那段说到的平均学分绩点也叫裸绩点，还有一个概念叫GPA。打比赛可以快速获得GPA，比如说拿到一个A类甲等国家一等奖就可以在裸绩点的基础上+1.5，国二是+1.0，国三/省一是+0.7，省二是+0.5，省三是+0.3.除此之外还有奖金可以拿，美美得吃了。A类乙等也可以加GPA，但是没有奖金；B类甲等GPA会降级（国家级按照省级），并且也没有奖金。打比赛还可以获得创新学分，每个同学需要2分的创新学分才能毕业，所以大家尽量早点拿到，不要影响毕业了。竞赛的相关内容会在（二）里详细展开。
  那么这个GPA有啥用嘞？当然是评奖学金。GPA会和德智体美劳等元素加起来组成综测，综测从夯到拉排名就是奖学金的顺序。综测文件因学院而异。
  每个学期都会评优秀学生奖学金，大致可以分成一等奖学金、二等奖学金、三等奖学金、单项奖学金。体测不过和挂科（补考过也没用）的同学都评不了。
  一等奖学金是1500元，占专业人数的5%，要求平均学分绩点排名列专业前10%，且学分绩点大于等于4
  二等奖学金是750元，占专业人数10%，要求平均学分绩点排名列专业的25%，且学分绩点大于等于3.5
  三等奖学金是400元，占专业人数15%，要求平均学分绩点排名列专业的45%，且学分绩点大于等于2.5
  单项奖学金包括学习进步奖、学生工作奖、文艺活动奖、道德风尚奖、社会实践奖、创新创业奖、体育活动奖等。
  然后每年都会评选省政府奖学金（6000元）。一般来说拿两次一等奖学金是很稳的，拿一次一等、一次二等也有一定机会。
  每年也会评选国家奖学金（10000元）。这个就比较困难了，一个学院也没有几个名额。如果你读大学的时候发现自己好像还不错，很有天赋，就可以抓好绩点的同时多搞一搞科研竞赛、社会实践。如果能拿到国家奖学金的话，保研考研申请学校、就业的时候都会有质的飞跃。
  除此之外还有国家励志奖学金，这个主要面向家庭情况没有那么好的同学。只要你愿意努力，国家就会帮助你！
  如果你的绩点还不错，就可以进行转专业。（只能转一次，所以想好再转！）这个每年政策都不太一样，在2025级应该是大一下的时候可以进行第一次跨学院的转专业。文科专业转理工科专业时，如果高考没有选择物理，就需要参加物理考试。
  保研名额大概会在大三下的时候确认。这个老学长也不知道要怎么战胜，这里就不献上焚决了。但是需要注意的是，保研只是给你这个名额，具体去哪个学校得你自己去联系好导师（也叫套磁）、自己去参加夏令营、自己去预推免。这个如果学长能上岸的话后续会更新的。直博的意思是直接攻读博士，好处是只需要读五年就可以拿到博士学院，坏处了如果延毕了或者不给你毕业的话那就只有本科学历了。

（二）竞赛篇
  竞赛篇真得好好讲一讲，初来乍到的小登往往会在竞赛上吃尽苦头。希望这篇焚决能帮助你快速上手竞赛。
  首先先按照角色定位来讲一讲吧。一个团队中一般需要有负责人、产品经理、研发岗、美工和答辩手组成，当然也可以身兼数职。
  有些比赛只能三个人，比如acm、电赛、数模；有些比赛可以15个人，比如说国创。不过需要注意的是比赛绩点基本上只加前五顺位的同学。当然如果硬要分也不是不行，但是一个国一的队伍只有1.5*5=7.5的绩点，如果要分给10个同学的话那每个人就只能有0.75的绩点。那可能就有人要问了：那我如果就只招3个人，岂不是每个人可以+2.5美美得吃？包不行的兄弟，加分上限就是我在上文说过的那个。（国一1.5，国二1.0，国三/省一0.7，省二0.5，省三0.3）
  负责人是最累的，需要协调全局，看好时间，做好分工。一般来说开发、文档、ppt、视频、答辩每个模块都需要参与。当一次负责人感觉自己要被超度了。
  产品经理的主要定位就是梳理需求，写好技术边界，然后写一篇prd文档交付给研发岗。注意一定要写好技术边界！不然有好几个研发同时在做一个东西，到时候merge（合并）到一起发现重复了然后疯狂报错你就老实了。
  研发岗无需多言，我们团队的研发大帝单独开了一个专栏给研发的同学，可以翻到后文去阅读一下。
  美工不是一个很简单的事情。创赛（创业类比赛，比如说国创赛、挑战杯这种）的ppt要求特别高，每页的内容不能太多、逻辑清晰、定位合理、观感要好...想做美工的同学可以到小红书等平台去逛一逛，应该有很多卖模板的，也可以找学长学姐要一下他们的比赛ppt。与此同时，像canva（可画）、figma、PS等软件都需要用的比较熟练，才能做好PPT，设计好UI（用户界面）、UX（用户体验）。如何写好技术文档我们也会在技能篇单独开一个专栏，可以期待一下。
  答辩手需要对项目比较了解，好的口才、自信的表现是必不可少的。一般来说答辩讲完ppt以后，评委会对项目做出产品与研发上的提问，所以答辩之前一定要做好准备。实在背不出来稿子的话，红色的卡纸手卡在打印店都可以让他帮你做。
  Alright以上就是我们一个team大致的分工，接下来我们就来讲一下具体竞赛的一个情况。老学长们打过很多比赛，下面会一一做介绍：

  1、数学建模（比赛当天发题，一般3-4天完成后提交）：
  数模是一个性价比比较高的比赛，因为数模的比赛特别多，包括数模的国赛（A类甲等）、数模的美赛（A类乙等）、数模的泰迪杯（B类甲等）等，相当于你一支队伍可以疯狂打数模加绩点。如果一个队伍拿到了数模国赛的一等奖，队伍内三名同学可以全员拿到保研资格，这在别的比赛中是想都不敢想的。
  数模队伍主要分成三个职务，建模手、代码手和论文手。当然大ai时代如果三个人都比较会用ai的话可以多借助一下ai神力。
  建模手需要比较强的问题定位与分析能力，一些基本的像遗传算法啊、粒子群算法啊、模拟退火算法等等都需要比较熟悉。借助ai神力的话可以让gpt，Gemini等做一下deep research，最后效果可能就会比较好。
  代码手主要负责将建模手的问题建模转化为代码的求解，并且画一些比较好看的图。需要对python比较熟悉，也可以用matlab（这个有很多数学的库）、R语言（这个画图比较好看）。当然用image-2、nanobanana 2K画，或者直接让ai用前端写比如说html文件，图的展示效果也会很不错。写代码的话国外的Claude效果很好，国内的glm（智谱）也不错。
  论文手主要负责写论文。个人建议最好学习一下使用LaTeX，然后在texstudio写，不要到overleaf这种在线的平台去编译避免论文泄露。当然如果你的word用的炉火纯青的话，使用word也可以。不管是word还是latex，都要记得最后一定要转成pdf再提交。不要忘了写ai使用报告，不要忘了写参考文献，不要超过限制页数，不要卡着提交时间提交。

  2、ACM：
  ACM是计算机领域认可度非常非常高的比赛（A类甲等），一个队也是三个人。这个比赛难度非常之高，需要大量的精力。不过如果能拿到acm的金牌这辈子也是有了。以前打过信奥的同学可以尝试一下，其他同学谨慎考虑。

  3、计算机设计：
  计算机设计是一个A类乙等比赛，相对于上面的两个比赛来说难度较低。这个比赛比较看重技术方面的创新，不是很看重商业上的表现。你需要比较多的专利、软著等证明你创新性的材料。

  4、浙江省服务外包：
  需要注意这是省级的A类甲等赛事，不是国服务外包的省赛！服务外包类竞赛都是企业发题，然后团队（五人）在几个月的时间内完成指定的功能，并作一定的创新。不建议从0开始做新项目，但是答辩的时候不能展示出你这是传家宝项目。
